# 坦克大战 (Tank Battle)

一个使用 HTML5 Canvas 制作的经典坦克大战游戏。

## 游戏说明

- **移动**：方向键 ↑ ↓ ← → 或 W A S D
- **射击**：空格键
- **暂停**：P 键

## 游戏规则

- 消灭所有敌方坦克即可过关
- 保护基地（底部旗帜标志），不能被敌人击中
- 砖墙可以被子弹摧毁，钢铁墙壁无法摧毁
- 每关 10 辆敌方坦克
- 玩家有 3 条生命

## 技术栈

- HTML5 Canvas
- 纯 JavaScript（无框架依赖）
- 单文件即可运行

## 运行方式

直接在浏览器中打开 `index.html` 即可。
